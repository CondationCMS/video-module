# Video module

This module i

## Integrate via shortcode

### In markdown 

```markdown
[[video type="youtube" id="y0sF5xhGreA" title="Everybody loves little cats" /]]
```

### In Template code

```html
[(${shortCodes.call('video', #{'type' : 'youtube', 'id': 'y0sF5xhGreA', 'title' : 'Everybody loves little cats'})})]
```
